# Angular Capacitor Starter

A project starter template using Angular and Capacitor,  
featuring core components focused on Compliance and Security based on **ISO 27001** principles.  

Designed to help developers build secure, compliant mobile applications for both **iOS and Android app stores**,  
addressing key regulatory and data protection requirements.

## About

This starter project provides foundational tools and components to help developers implement ISO 27001-aligned security and compliance controls in mobile apps built with Angular and Capacitor.
It also incorporates best practices to meet app store compliance requirements for both iOS and Android platforms.
