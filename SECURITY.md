# Security Policy

## Supported Versions

The following table lists the versions currently supported with security updates:

| Version | Supported          |
| ------- | ------------------ |
| 0.0.0   | ✅ Supported       |
| < 0.0   | ❌ Not Supported   |

Only actively maintained versions receive security fixes. We recommend upgrading to the latest supported version to ensure continued security compliance.

---

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it **privately** to prevent potential abuse.

**Contact:**  
~~📧 Email: [security@bloxxyyy.dev]~~
🔗 GitHub: [https://github.com/bloxxyyy](https://github.com/bloxxyyy)

Please include the following in your report:

- A clear description of the vulnerability
- Steps to reproduce (if applicable)
- Affected versions
- Potential impact

### Response Timeline

We aim to acknowledge all security reports within **1 month**. If the report is verified and accepted:

- You will be notified once the fix is released.
- You may be credited in the changelog unless anonymity is requested.

If the report is not accepted, we will provide a reasoned response.

> Please do **not** disclose security issues publicly until we have reviewed and addressed them responsibly.

Thank you for helping improve the security and reliability of this project.
